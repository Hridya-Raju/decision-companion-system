How this Decision System Works

1. Options

Enter the items that you want to compare. Separate them using commas.
Example: Laptop A, Laptop B

2. Criteria

Enter the criterias based on which you compare your options.
Example: Price, Battery, Memory

3. Criteria Type

Write min if smaller value is better, max if bigger value is better.
Example: If the criterias are Price, Battery, Memory; as lower price is generally preferred
while higher battery life and higher memory is preferred, the inputs will be like:
min, max, max

4. Values

Give scores row-wise for each option.
Example: The price of first item is 55000, battery life is 25 hours, memory is 8GB.
The price of second item is 70000, battery life is 32 hours, memory is 16GB.
Then input is given as:
55000 25 8 | 70000 32 16

5. Weightage

It gives the importance of each criterion.
Example: Importance of Price, Battery, Memory are given as (on a scale of 10)
7, 5, 8

The system normalizes values, multiplies by weights, and ranks options based on total score.
